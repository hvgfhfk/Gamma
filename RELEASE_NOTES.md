# 릴리즈 노트

## 버전 1.0.0.0 (2026-01-25)

### 주요 기능
- ✅ 감마, 밝기, 대비 실시간 조정
- ✅ NVIDIA 제어판과 동일한 계산 방식
- ✅ 단축키 지원 (기본값/변경값 전환)
- ✅ 프로세스 자동 감지 및 설정 적용
- ✅ 프로세스별 단축키 설정
- ✅ Windows 시작 시 자동 실행
- ✅ 시스템 트레이로 최소화
- ✅ 포터블 애플리케이션 (설치 불필요)
- ✅ 종합 오류 로깅

### 기술 사항
- Windows GDI32 API 사용
- NVIDIA NvAPI 지원 (선택적, HDR 모드에서도 동작)
- MVVM 패턴 적용
- JSON 기반 설정 저장

### 알려진 제한사항
- 다중 모니터 환경에서는 기본 디스플레이만 지원
- HDR 모드에서 GDI32 API는 제한적으로 동작 (NVIDIA NvAPI 사용 시 해결)
- 일부 시스템에서는 관리자 권한 필요

### 시스템 요구사항
- Windows 7 SP1 이상
- .NET Framework 4.7.2 이상
