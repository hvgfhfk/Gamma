# Gamma (감마 컨트롤러)

Windows용 **모니터 감마·밝기·대비** 조절 프로그램입니다. 게임/프로그램별로 다른 화면 설정을 자동 적용하고, 아이템 목록·즐겨찾기·메타포지 거래 연동 등 편의 기능을 제공합니다.

![메인 화면](docs/screenshot-main.png)

---

## ✨ 주요 기능

### 🖥️ 감마·밝기·대비 조절
- **감마**: 0.3 ~ 4.4 범위에서 슬라이더로 조절 (어두움 ↔ 밝음)
- **밝기**: -100% ~ 100% 조절
- **대비**: RGB 채널별 대비 조절
- 선택한 **모니터에만** 적용 가능 (다중 모니터 지원)

### 📌 프로세스별 자동 적용
- 특정 프로그램(예: 게임 실행 파일)이 실행되면 **해당 프로그램용 감마/밝기/대비**를 자동 적용
- 프로그램 종료 시 기본값으로 복원
- 여러 프로세스 규칙을 등록해 두고 필요 시 켜기/끄기 가능

### ⌨️ 단축키
- **기본 프로필** / **사용자 지정 프로필**에 단축키를 지정해 즉시 전환
- 게임 중에도 전역 단축키로 화면 밝기 빠르게 변경

### 📋 아이템 목록 & 즐겨찾기
- 아이템 목록 검색, 즐겨찾기 저장
- **선택한 아이템 오버레이**: 지정한 단축키로 게임 화면 위에 선택한 아이템만 띄워 보기

### 🔗 Metaforge (Arc Raiders) 연동
- [Metaforge](https://metaforge.app) Arc Raiders 마켓 거래 목록 조회
- 판매/구매 목록 확인 및 채팅 연동

### ⚙️ 기타
- **다국어**: 한국어 / English
- **시작 프로그램** 등록 옵션
- **트레이** 최소화, 단일 인스턴스 실행
- **자동 업데이트** (GitHub Releases 연동)

## 📋 요구 사항

- **OS**: Windows 10 / 11 (64비트 권장)
- **.NET**: .NET Framework 4.8
- **디스플레이**: GDI 감마 램프를 지원하는 GPU/드라이버 (대부분의 PC에서 지원)

## 🚀 설치 및 실행

### 1. 릴리스에서 설치 (권장)
1. [Releases](https://github.com/hvgfhfk/Gamma/releases) 페이지에서 최신 버전 확인
2. `Gamma-vx.x.x.zip` (또는 설치용 패키지) 다운로드
3. 압축 해제 후 **Gamma.exe** 실행
